# 🎮 Tic Tac Toe

A clean, minimal Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, just pure fun.

![Game Preview](https://img.shields.io/badge/status-playable-brightgreen?style=flat-square) ![HTML](https://img.shields.io/badge/HTML-5-orange?style=flat-square&logo=html5) ![CSS](https://img.shields.io/badge/CSS-3-blue?style=flat-square&logo=css3) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow?style=flat-square&logo=javascript)

---

## ✨ Features

- 🟦 Two-player gameplay (X vs O)
- 🔄 Instant restart button to play again
- 📢 Live status updates showing whose turn it is
- 🎨 Clean UI with the [Outfit](https://fonts.google.com/specimen/Outfit) font from Google Fonts
- ⚡ Zero dependencies — runs straight in the browser

## 🚀 Getting Started

No installation needed. Just open the game in your browser:

**Option 1 — Clone and open locally:**
```bash
git clone https://github.com/084divyanshuraj/tic-tac-toe.git
cd tic-tac-toe
# Open index.html in your browser
```

**Option 2 — Direct download:**

Download the ZIP from the [repository](https://github.com/084divyanshuraj/tic-tac-toe), unzip it, and open `index.html`.

## 🕹️ How to Play

1. The game starts with **Player X**
2. Players take turns clicking an empty cell to place their mark
3. The first player to get **3 in a row** (horizontally, vertically, or diagonally) wins!
4. If all 9 cells are filled with no winner, it's a **draw**
5. Click **Restart Game** to play again
